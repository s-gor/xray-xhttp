# 06. Клиент

На этом этапе настраиваем клиентское подключение к серверу.

## Что уже должно быть готово

- домен настроен;
- TLS-сертификат получен;
- Xray установлен;
- XHTTP настроен;
- сервер Xray запущен без ошибок.

## Клиенты

Можно использовать:

- v2rayN для Windows;
- v2rayNG для Android;
- другой клиент с поддержкой нужного транспорта.

## Какие данные нужны для клиента

Для подключения обычно нужны:

- адрес сервера;
- порт;
- UUID;
- путь XHTTP;
- SNI / server name;
- параметры TLS.

## Проверка сервера

На сервере проверить статус Xray:

systemctl status xray

Посмотреть последние логи:

journalctl -u xray -n 50 --no-pager

## Проверка клиента

После добавления профиля в клиенте нужно проверить:

- подключается ли клиент;
- есть ли интернет через профиль;
- нет ли ошибок в логах Xray;
- совпадают ли UUID, домен, путь и TLS-настройки.

## Важно

Если клиент не подключается, сначала проверить простые вещи:

- домен указывает на правильный IP;
- порт открыт;
- Xray запущен;
- в клиенте указан правильный UUID;
- SNI совпадает с доменом;
- путь XHTTP совпадает с серверной конфигурацией.
