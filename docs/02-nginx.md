# 02. Nginx

На этом этапе устанавливаем Nginx и проверяем, что веб-сервер отвечает на HTTP-запросы.

## Установка

apt update

apt install nginx -y

## Проверка сервиса

systemctl status nginx

Сервис должен быть в состоянии:

active (running)

## Проверка порта 80

ss -tulpn | grep :80

Должно быть видно, что Nginx слушает порт 80.

## Проверка снаружи

С компьютера можно открыть в браузере:

http://example.com

или проверить через curl:

curl -I http://example.com

В ответе должен быть HTTP-заголовок от Nginx.

## Важно

Перед получением сертификата Let's Encrypt порт 80 должен быть доступен извне.

Если на сервере уже запущен другой сервис на порту 80, Nginx не сможет стартовать.
