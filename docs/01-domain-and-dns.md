# 01. Домен и DNS

На этом этапе нужно подготовить домен и направить его на IP-адрес сервера.

## Что должно быть

- куплен или уже доступен домен;
- известен публичный IP-адрес VPS;
- создана DNS A-запись домена на IP сервера.

## Пример

Если сервер имеет IP 203.0.113.10, то DNS-запись должна быть примерно такой:

example.com A 203.0.113.10

## Проверка DNS

На своём компьютере можно проверить:

nslookup example.com

или:

dig example.com

В ответе должен быть IP-адрес сервера.

## Важно

Перед получением TLS-сертификата домен уже должен указывать на сервер.

Также порт 80 должен быть доступен снаружи, иначе Let's Encrypt не сможет проверить домен.
