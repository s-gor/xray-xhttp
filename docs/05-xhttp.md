# 05. XHTTP

На этом этапе настраиваем XHTTP transport для Xray.

## Что уже должно быть готово

- домен указывает на сервер;
- Nginx установлен;
- TLS-сертификат получен;
- Xray установлен;
- файл /usr/local/etc/xray/config.json доступен для редактирования.

## Пример конфигурации

Пример Xray-конфигурации находится здесь:

examples/xhttp-config.json

Перед использованием нужно заменить тестовые значения:

- example.com
- YOUR_UUID
- YOUR_PATH

## Проверка конфигурации

После изменения конфига проверить синтаксис:

xray run -test -config /usr/local/etc/xray/config.json

Если ошибок нет, перезапустить Xray:

systemctl restart xray

## Проверка сервиса

systemctl status xray

## Логи

journalctl -u xray -n 50 --no-pager

## Важно

Если Xray не стартует, сначала проверить JSON-синтаксис и логи.

На этом этапе важно не смешивать несколько вариантов конфигурации сразу. Сначала нужно добиться запуска одного рабочего варианта.
